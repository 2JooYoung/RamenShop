## About

RamenShop은 라면 상품을 소개하고 판매하는 콘셉트의 웹페이지입니다.

- 섹션을 나누어 메인 페이지 레이아웃 구성
- 상품 카드(이미지, 이름, 가격) UI 만들기
- 버튼, 네비게이션 등 기본적인 컴포넌트 스타일링
- JavaScript 로 동적인 요소 구현 (예: 버튼 클릭, 스크롤, 이벤트 처리 등)
